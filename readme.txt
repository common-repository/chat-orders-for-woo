== Installation ==

### INSTALL "ORDER WOO CHAT" FROM WITHIN WORDPRESS

1. Visit the plugins page within your dashboard and select ‘Add New’;
1. Search for ‘Orders Woo Chat’;
1. Activate Woo Orders WhatsApp from your Plugins page;

### INSTALL "ORDER WOO CHAT" MANUALLY

1. Upload the ‘order-woo-chat’ folder to the /wp-content/plugins/ directory;
1. Activate the "Woo Orders By Wha" plugin through the ‘Plugins’ menu in WordPress;

== Upgrade ==

Just click “Update” on the plugins page and let WordPress do it for you automatically.
